name: entrevistame
---

Entrevístame exhaustivamente sobre cada aspecto de este plan hasta que lleguemos a un entendimiento común. Analice cada rama del árbol de diseño, resolviendo las dependencias entre las decisiones una por una. Para cada pregunta, proporcione su respuesta recomendada.

Formule las preguntas una a una, esperando retroalimentación antes de continuar. Hacer varias preguntas a la vez puede resultar confuso.

Si una pregunta puede responderse explorando el código fuente, explore el código fuente.
